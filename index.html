Version Control

Version Control

Version control is a useful skill to have at any company where multiple revisions need to be made on a project and where there are many people working on the same thing at once. When writing computer programs, everything is dependent on the tiniest detail and one small error could create total chaos. With version control, every change is tracked and each version is saved so team members can more easily fix bugs and find the best and most efficient way to implement their program.


I use git & Github to manage my solo projects but also have experice using Team Foundation Server in a group setting while working on Job PDX

.NET Framework

.NET Framework

The .NET Framework is Microsoft's collection of tools and infrastructure that allow developers to use several different languages and compile them all into a platform-neutral intermediate language so that it can run within the CLR on any Windows machine. As Microsoft is a fairly relevant company in Seattle, I can definitely see myself using the .NET Framework quite a bit in the future.


I believe I would really enjoy using C# and ASP.NET in my future career as a Software Developer, and I'm very proud to be able to say it's one of my skills.
